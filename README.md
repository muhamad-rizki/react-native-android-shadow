# react-native-android-shadow

React Native Shadow View for Android

## Installation

```sh
npm install react-native-android-shadow
```

## Usage

```js
import { AndroidShadowView } from "react-native-android-shadow";

// ...

<AndroidShadowView color="tomato" />
```

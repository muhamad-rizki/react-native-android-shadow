# @vesselsoft/react-native-shadow

React Native Shadow View for Android

## Installation

```sh
npm install @vesselsoft/react-native-shadow
```

## Usage

```js
import { AndroidShadowView } from "@vesselsoft/react-native-shadow";

// ...

<AndroidShadowView color="tomato" />
```
